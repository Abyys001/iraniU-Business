# فونت‌ها

## Yekan Bakh (فایل‌های محلی)

خانوادهٔ **Yekan Bakh** از فایل‌های OTF در همین پوشه بارگذاری می‌شود:

| فایل | وزن CSS |
|------|---------|
| `YekanBakh-Regular.otf` | ۴۰۰ |
| `YekanBakh-Bold.otf` | ۷۰۰ |
| `YekanBakh-ExtraBlack.otf` | ۸۰۰ و ۹۰۰ |

تعریف `@font-face` در [`css/styles.css`](../css/styles.css) و متغیر `--font` است.

فایل‌های `.ttf` همان خانواده در سیستم اختیاری‌اند؛ برای وب همین سه فایل OTF کافی است.

## رنگ برند

رنگ اصلی برند در CSS: `#3a0b47` (متغیر `--color-primary`).
