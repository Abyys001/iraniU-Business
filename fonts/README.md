# فونت‌ها

## فونت اصلی سایت

متن‌ها با **Vazirmatn** (مناسب فارسی و RTL) از [Google Fonts](https://fonts.google.com/specimen/Vazirmatn) بارگذاری می‌شوند؛ لینک در `<head>` همه صفحات HTML است.

اگر بخواهید بدون وابستگی به اینترنت کار کنید، می‌توانید همان خانواده را خودتان میزبانی کنید و `@font-face` را در [`css/styles.css`](../css/styles.css) اضافه کنید؛ در آن صورت متغیر `--font` را با نام خانواده جدید هماهنگ کنید.

## رنگ برند

رنگ اصلی برند در CSS: `#3a0b47` (متغیر `--color-primary`).
